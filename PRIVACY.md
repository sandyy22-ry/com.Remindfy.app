Privacy Policy for Remindfy

Short summary:
- We collect no personal data by default. Local reminders, notes, and photos are stored only on the user's device unless the user explicitly enables any sync feature.
- We use device notifications and exact alarms to deliver reminder alerts. These features require runtime notification permission and the SCHEDULE_EXACT_ALARM permission on Android.
- We use internet access only for optional features (if any) and do not transmit personal reminders without explicit consent.
